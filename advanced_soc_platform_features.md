# Advanced SOC Platform Features

This file is aspirational. It describes possible future capability areas and must not be confused with current verified repo behavior.

## Possible future directions
- deeper case graph correlation across repeated indicators
- richer approval workflows with stronger actor policy checks
- live collaboration previews with stronger redaction controls
- operator-focused terminal/TUI drill-down views
- expanded runtime forensics and health telemetry
- more complete audit trails and replayable demo scenarios

## Constraint
Any future feature must preserve the current project rules:
- honest scope
- bounded operator output
- production-safe defaults
- repo-bound implementation truth
